# CampusRPG

## 项目简介

CampusRPG 是一个使用 C++ 开发的校园 RPG 冒险游戏系统。

## 主要功能

- 角色管理
- 背包管理
- 商店系统
- 任务系统
- 战斗系统
- 等级成长
- 存档与读取

## 技术亮点

本项目使用 STL 高级应用作为挑战任务：

- 使用 vector 管理背包物品、任务列表和敌人列表
- 使用 unordered_map 管理商店商品
- 使用 algorithm 中的 sort 对敌人按难度排序

## 开发环境

- Visual Studio
- C++
- GitHub
